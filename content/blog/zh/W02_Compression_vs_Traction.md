# 压缩式气弹簧与牵引式气弹簧：区别是什么？

当工程师或设计师提到“气弹簧”时，他们通常指的是标准的**压缩式气弹簧**。然而，还有一种工作方式完全相反的类型：**牵引式气弹簧**（也称为拉动式气弹簧）。

选择错误的类型可能导致机构无法正常工作，甚至造成安全事故。在本文中，我们将厘清两者的区别，解释各自的工作原理，并帮助你确定项目所需的气弹簧类型。

## 1. 压缩式气弹簧（“推”型）

![压缩式气弹簧](/images/products_page/compress_gas_spring.jpg)

这是最常见的气弹簧类型，应用于90%的场景，例如汽车引擎盖、厨房橱柜和机器外壳。

- **工作原理**：在静止（自由）状态下，活塞杆处于**完全伸出**状态。当你将活塞杆推入时，内部氮气被压缩，产生一股**推**力，将活塞杆重新推出。

- **功能**：提供提升辅助，帮助你打开沉重的盖子并保持其处于打开状态。

- **典型应用**：
  - 打开汽车后备箱
  - 抬起床架（储物床）
  - 支撑工业舱口

**核心特点**：需施加外力才能*压缩*它，其自身会自动伸出。

## 2. 牵引式气弹簧（“拉”型）

![牵引式气弹簧](/images/product/gas-spring/traction-gas-spring/traction-gas-spring.jpg)

牵引式气弹簧虽不常见，但在特定场景中至关重要——当你需要将物体拉合或保持在缩回位置时，它能发挥关键作用。

- **工作原理**：在静止（自由）状态下，活塞杆处于**完全缩回**状态（位于缸体内）。内部压力会将活塞杆保持在缸体内部，使用时需将活塞杆拉出；松开后，弹簧力会将活塞杆**拉**回缸体内。

- **功能**：提供闭合辅助或张力，保持盖子闭合，或帮助拉起沉重的坡道。

- **典型应用**：
  - 车库门（辅助拉升闭合）
  - 闭合需要密封紧密的重型舱口
  - 张紧传送带

**核心特点**：需施加外力才能*伸出*它，其自身会自动缩回。

## 对比表格

| 特性 | 压缩式气弹簧 | 牵引式气弹簧 |

| :--- | :--- | :--- |

| **静止状态** | 活塞杆伸出 | 活塞杆缩回 |

| **力的方向** | 向外推（伸出） | 向内拉（缩回） |

| **主要用途** | 提升 / 开启 | 闭合 / 张紧 |

| **标识** | 通常标有指向外部的箭头 | 通常标有指向内部的箭头 |

| **成本** | 标准款，成本较低 | 专用款，成本较高 |

## 如何选择合适的类型？

**场景A：你有一个沉重的盖子，需要将其抬起。**

> 你需要**压缩式气弹簧**。该弹簧会在你抬起盖子时抵消重力，提供辅助力。

**场景B：你有一个会自行打开的坡道或门，希望它能自动闭合或更易拉合。**

> 你可能需要**牵引式气弹簧**。该弹簧会对抗重力，将物体拉回闭合位置。

## 总结

- **压缩式** = 推力（活塞杆自然伸出）。

- **牵引式** = 拉力（活塞杆自然缩回）。
